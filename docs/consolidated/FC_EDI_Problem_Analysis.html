1.5 The Tally Problem

Tally Prime dominates Indian MSME accounting with 70-90% market share and 92% user satisfaction. It supports GST e-invoicing, e-Way bills, and exports to XML, CSV, JSON, and PDF. However, Tally cannot generate EDI X12 or EDIFACT documents natively. There is no built-in AS2 protocol support, no EDI mapping engine, and no VAN connectivity. The gap between what Tally outputs (XML/CSV) and what global buyers require (EDI 850/810/856) is the core problem FactoryConnect solves.

1.6 Ariba/Coupa Onboarding Pain for MSMEs

SAP Ariba gives suppliers just 3 weeks to complete registration with up to three questionnaires. Subscription tiers range from free (≤5 docs/year) to $750/year for Silver. Transaction fees of 0.155% apply, capped at $20,000/year per buyer. For an Indian MSME processing $500K in orders, that's ~$775/year in Ariba fees alone — plus the technical burden of understanding cXML, configuring document routing, and maintaining compliance. Coupa's supplier portal is free but the buyer-side implementation costs $25K–$1.5M, meaning buyers push compliance costs downstream to suppliers.

1.7 EU CBAM Reporting Burden

The EU Carbon Border Adjustment Mechanism entered its definitive phase in January 2026, affecting steel, aluminum, cement, fertilizers, electricity, and hydrogen exports. India has ~$8 billion in CBAM-exposed exports to the EU. Key facts:

Manual carbon calculation per shipment requires collecting data from dozens of small upstream vendors — a process that can take 8-16 hours per shipment on spreadsheets.

Each FactoryConnect component maps directly to a specific pain point. The architecture was designed ground-up for Indian MSME constraints.

🔴 Tally can't generate EDI

70-90% of Indian SMEs use Tally. It exports XML/CSV but not X12/EDIFACT. Manual re-keying is the only option.

→
✅ Bridge Agent → Tally Integration

Windows desktop agent connects directly to Tally's XML Server API. Auto-extracts invoices, POs, inventory into canonical format. Zero manual export. Runs silently in system tray.

🔴 Every buyer needs different formats

Walmart wants X12 over AS2. H&M wants EDIFACT. Ariba wants cXML. Each requires separate mapping at $2K-$5K/partner.

→
✅ Canonical Data Model

One integration from Tally → canonical format. FactoryConnect translates to any buyer format (X12, EDIFACT, cXML, Coupa). Add new buyers without new integrations.

🔴 EDI mapping takes weeks, costs $10K+

Traditional EDI requires field-by-field mapping by specialists. 6-8 weeks per trading partner. Testing and certification adds more time.

→
✅ AI-Assisted Field Mapping

Machine learning maps Tally fields to buyer EDI segments automatically. Human review on exceptions only. Reduces mapping from weeks to hours. Pre-built templates for top 20 buyers.

🔴 Fear of going live — what if it breaks?

MSMEs can't risk order disruption. One failed EDI transmission could trigger chargebacks or lost orders. Fear prevents adoption.

→
✅ Shadow Mode

Run FactoryConnect in parallel with existing manual process. Compare outputs side-by-side. Validate accuracy before switching. Zero-risk transition — go live only when confidence is 100%.

🔴 CBAM reporting: 8-16 hrs/shipment manually

Product-level emissions data from dozens of vendors. Spreadsheet calculations. Default values carry 10-30% penalty markups. €100/ton for reporting errors.

→
✅ CBAM Compliance Module

Automated carbon reporting alongside trade documents. Calculates actual emissions from production data in Tally. Generates EU-compliant CBAM declarations. Avoids default value penalties — saves €10-20/ton.

🔴 Messages lost, no delivery guarantee

Email-based workflows: no acknowledgment, no retry, no audit trail. EDI via VAN: expensive failover. One lost ASN = chargeback.

→
✅ Saga Coordinator + Transactional Outbox

Guaranteed delivery with retry and circuit breaker patterns. Transactional outbox ensures zero message loss — if it's in the database, it will be delivered. Full audit trail for every document.

🔴 Enterprise security too expensive for MSMEs

EDI compliance requires data isolation, audit trails, encryption. Enterprise solutions cost $50K+/year. MSMEs can't afford it.

→
✅ RLS Multi-Tenancy

Row-Level Security ensures complete tenant isolation at the database level. Enterprise-grade security at MSME pricing. Each factory's data is cryptographically isolated. SOC 2 compliant architecture.
